# Цели
В данном проекте было необходимо:
  - создать примитивную страницу со списком новостей (Тексты и изображения новостей произвольные (изображения должны быть загружены из интернета, например с сайта unsplash.com)) 
  - создать собственные разработанные стили для сайта, используя систаксис CSS
  - потренироваться со стилями и изменить некоторые компоненты на сайте, а именно:
    1) Используя селектор псевдокласса задать цвет фона у каждой второй (четной) новости (цвет на ваше усмотрение)
    2) Используя селектор псевдоэлемента добавить к заголовкам пeрвых 2х новостей слово "New"

## Используемые технологии

* HTML
* CSS


## Как открыть/запустить

1. Локально - склонировать репозиторий, зайти в папку с проектом сайта в файловом менеджере и открыть файл index.html.
2. Через локальный сервер - склонировать репозиторий, зайти в папку с проектом сайта в файловом менеджере и в редакторе кода запустить проект через локальный сервер (Live Server например).
